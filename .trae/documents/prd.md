
## 1. Product Overview
苹果官网风格的个人博客网站，提供极简美学设计与流畅的交互体验，展示个人文章与创作内容。

## 2. Core Features

### 2.1 User Roles (if applicable)
| Role | Registration Method | Core Permissions |
|------|---------------------|------------------|
| 访客 | 无需注册 | 浏览所有文章、页面和评论 |

### 2.2 Feature Module
1. **首页**: Hero大图区域、最新文章卡片网格、导航栏
2. **文章列表页**: 分类筛选、标签筛选、搜索功能、分页
3. **文章详情页**: 文章内容展示、目录导航、相关推荐
4. **关于页**: 个人介绍、作品展示
5. **留言页**: 留言表单、留言列表
6. **深色/浅色模式切换**: 一键切换主题

### 2.3 Page Details
| Page Name | Module Name | Feature description |
|-----------|-------------|---------------------|
| 首页 | Hero区域 | 全屏大图、滚动视差效果、鼠标跟随微光 |
| 首页 | 文章卡片网格 | 文章卡片悬浮效果、平滑淡入动画 |
| 导航栏 | 粘性导航 | 滚动时变为半透明磨砂玻璃 |
| 文章详情页 | 文章内容 | 优雅排版、目录导航、动画过渡 |
| 所有页面 | 主题切换 | 深色/浅色模式一键切换、平滑动画 |

## 3. Core Process
用户访问博客 → 浏览首页Hero区域和最新文章 → 点击文章卡片进入详情页 → 阅读文章并查看相关推荐 → 可通过导航栏访问其他页面 → 可切换深色/浅色模式

## 4. User Interface Design
### 4.1 Design Style
- **主色调**: 纯白/纯黑，浅灰、深空蓝点缀
- **按钮风格**: 精致圆角、轻微阴影、悬浮效果
- **字体**: 无衬线细体（SF Pro风格），字重层级清晰
- **布局风格**: 大量留白、卡片网格、居中对齐
- **质感**: 磨砂玻璃、轻微阴影、卡片悬浮

### 4.2 Page Design Overview
| Page Name | Module Name | UI Elements |
|-----------|-------------|-------------|
| 首页 | Hero区域 | 全屏大图、淡入动画、鼠标视差 |
| 首页 | 文章卡片 | 圆角卡片、悬浮效果、平滑加载 |
| 导航栏 | 粘性导航 | 磨砂玻璃效果、滚动变化 |
| 文章详情页 | 文章内容 | 优雅排版、目录侧边栏 |

### 4.3 Responsiveness
- 桌面端优先设计
- 完美适配移动端（响应式布局）
- 触摸优化

### 4.4 Animation Effects
- 滚动视差 + 序列帧动画
- 平滑滚动与导航
- 卡片/按钮悬浮效果
- 页面转场动画
- 骨架屏加载
